# Contributing to Tman

We love your input! We want to make contributing to Tman as easy and transparent as possible.

## How to Contribute

1. **Report Bugs**: Open an issue with a clear description and steps to reproduce.
2. **Suggest Features**: Open an issue describing the feature and why it would be useful.
3. **Submit Changes**:
   - Fork the repository.
   - Create a feature branch (`git checkout -b feature/amazing-feature`).
   - Commit your changes (`git commit -m 'Add amazing feature'`).
   - Push to the branch (`git push origin feature/amazing-feature`).
   - Open a Pull Request.

## Development Setup

- Install [Rust](https://www.rust-lang.org/tools/install)
- Install [Node.js](https://nodejs.org/)
- Install [Tauri dependencies](https://tauri.app/v1/guides/getting-started/prerequisites)
- Run `npm install` to install frontend dependencies.
- Run `npm run tauri dev` to start development mode.

## Coding Standards

- Follow Rust and Svelte best practices.
- Ensure your code is formatted and linted.
- Add comments where necessary.

Thank you for contributing!
